## [1.0.0] - 2025-12-16
### Added
- Initial stable release.
- Declarative testing Scenarios for Laravel + Pest.
- Support for Laravel 11 & 12 via Testbench 9/10.
- Support for Pest 3 & 4.
- API Route, Web Route, Policy, Form Request, Rule, Model and Command Scenarios and Contexts.
- Optional Docker environment + Laravel Workbench support for local development.
- CI using GitHub Actions + Act compatibility.
- PHPStan type-safety checks.
- 100% test coverage.
